Projeto realizado em dupla para avaliação da sessão 5 do módulo de fundamentos no curso de Desenvolvimento Web da **Trybe**.

* Flexbox
* Responsivo
* Forms
* Resolução de Conflitos no GIT
* Empatia
* Trabalho em Equipe
